# 修复与功能记录

## 功能新增

1. **邮箱注册登录** — 支持邮箱+验证码注册、邮箱+密码登录、邮箱找回密码，163 SMTP 发送验证码
2. **产品定价管理** — 管理后台新增"产品定价"标签页，支持在线编辑充值选项、订阅套餐、模型单价、签到奖励、分销配置等
3. **类型与风格后台管理** — 首页的类型和风格从后台动态加载，管理员可在"类型与风格"标签页增删改，封面图支持上传到MinIO

## Bug 修复

1. **默认模型错误** — 默认模型从 gemini-2.5 改为 gemini-3.1-flash-image-preview
2. **历史记录图片加载失败** — MinIO 内部地址(minio:9000)未转换为前端可访问的相对路径(/storage/)
3. **系统提示词泄露** — 用户中心和聊天记录中过滤掉英文系统提示词，只显示用户输入
4. **尺寸显示异常** — 宽高为空时显示"自动"而非空白
5. **中文生图指令缺失** — Gemini 生图添加 systemInstruction 强制输出中文文字
6. **消费记录提示词错误** — worker 保存用户原始中文提示词，而非增强后的英文提示词
7. **Token 刷新失效** — 前端 401 拦截器自动刷新 token 并重试请求，支持并发请求排队
8. **订阅激活失败** — 支付成功后正确设置 subscription_plan 和过期时间
9. **签到积分类型错误** — 签到奖励改为临时积分(gift_points)，当日 23:59 过期
10. **过期积分未清理** — 添加定时任务每 30 分钟清理过期的 gift_points
11. **支付宝回调漏洞** — 未配置支付宝时拒绝所有回调，而非无验证直接通过
12. **UTC 时区偏移 8 小时** — 全局修复前端时间解析，后端 UTC 时间补 Z 后缀
13. **生图任务重复执行** — 前端添加发送锁+提前设置 isGenerating，后端添加 5 秒去重窗口
14. **SQL 迁移顺序错误** — email 列 COMMENT 语句移到 ALTER TABLE 迁移块内执行
15. **验证码存储外键冲突** — 发送验证码改用内存缓存，避免 user_auth 表空 user_id 外键报错
16. **历史记录点击无反应** — 非首页点击历史记录后自动跳转到首页显示聊天内容
17. **历史记录图片不显示** — 前端+后端双重归一化 MinIO 内部 URL 为 /storage/ 路径
18. **移动端无历史记录** — 移动端侧边栏添加历史记录列表
19. **提示词反复重复** — _compose_image_model_prompt 将用户输入和生成计划拼接导致同一提示词出现多次，改为检测重复只保留一份
20. **场景库标签被截断** — 分类标签区域 max-w-800px + 滚动条隐藏导致最后标签显示不全，改为 flex-wrap 换行显示
21. **后台场景库保存失败** — syncScenesToServer 未携带 Authorization token 导致 401，修复后场景增删改正常同步到数据库
22. **默认尺寸非auto** — 初始宽高从 2048x2048/1024x1024 改为 0(auto)，让模型自动决定尺寸
23. **类型/风格显示反了** — 类型改为纯文字标签(无图片)，风格改为带封面图卡片，后端和管理后台同步调整
24. **尺寸选择器auto误显为自定义** — width=0时0/0=NaN导致fallback到custom，增加0值判断
25. **风格封面图缺失+裁剪** — 写实/动漫/3D补上封面图，图片从正方形裁切改为自适应完整显示
26. **多图创作页与首页不一致** — 多图创作页类型改为纯文字标签、风格改为带封面图卡片，数据从API动态加载，与首页保持一致
27. **历史记录标题被覆盖** — save_assistant_response 用AI回复覆盖了用户原始标题，去掉错误的标题更新逻辑
28. **纯对话无历史标题** — 流式聊天和图片模型的对话回复完成后也自动调用 summarizeSession 生成标题
29. **历史消息重复保存** — 非流式对话后端+前端各存一次助手消息，save_message 增加5秒去重检查
30. **历史图片被裁剪为正方形** — 用户中心、我的作品、创作记录、对话历史、聊天消息五处图片展示统一去掉 aspect-square + object-cover，按真实比例完整显示
31. **选择尺寸比例无效出图为方形** — handle_single_generate 中 LLM 分析提示词覆盖了用户选择的宽高，改为用户明确指定尺寸时跳过 LLM 分析直接使用
32. **复制按钮点击后无法粘贴** — navigator.clipboard.writeText 需要 HTTPS，HTTP 环境下静默失败；新增兼容工具 clipboard.ts，自动 fallback 到 execCommand('copy')，全局8处统一替换
33. **类型/风格封面图需手动输入URL** — 管理后台类型和风格封面图改为支持点击上传，后端新增 upload-cover 接口上传到 MinIO；类型也支持封面图，前端自动判断显示模式
34. **消费记录不写入（严重）** — worker._is_valid_url 只接受 http/https 开头的 URL，但 minio_url_prefix 默认为 /storage，图片 URL 为 /storage/xxx.png 被判定无效，settle_frozen_points 永不执行。线上165次冻结0次结算
